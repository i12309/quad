Необходимо добавить возможность вызова контекстного меню при клике на квадратик, правой клавишей мыши. 
Наполнение контекстного меню зависит от типа выбранной игры. 
Для этого в класс BaseModule добавить абстрактный метод вызова контекстного меню. 
Для каждого типа игры GameOfLife или PhysicsSimulation сделать вызов своего контекстного меню. 
И сделать наполнение этого меню следующим образом: 
1. PhysicsSimulation - контекстное меню содержит пункты выбора какого квадратики создать. 
1.1. Светло-серые - созданный квадратик будет падать вниз. 
1.2. Зеленый - квадратик будет падать вверх 
1.3. Синий - будет стоять на установленном месте. 
Если кликать на квадратики пока игра еще не запущена, левой клавишей мыши, то цвет квадратика меняется по кругу - серый, синий, зеленый и т.д.  
2. GameOfLife 
При выборе это игры все квадратики становятся одинакового светло-серого цвета. 
Контекстное меню пустое.
